# BC2406 Misinformation Detection

Individual assignment for BC2406 Analytics I: Visual and Predictive Techniques, focused on detecting misinformation using classification techniques.

## Overview
An end-to-end exploratory data analysis as well as predictive modelling using Logistic Regression and Decision Tree to indentify misinformation using a social media dataset provided in the assignment.
